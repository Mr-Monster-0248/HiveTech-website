Les Exercices

Exercice 1 - Memory

Un exercice de mémoire. Le patient doit retrouver les paires de cartes identiques, en retournant les cartes deux par deux.

Comporte plusieurs niveaux de difficulté.

Exercice 2 - Opérations

Un exercice de calcul. Le patient doit retrouver le bon résultat d'une opération, pour une opération simple.

Comporte plusieurs niveaux de difficulté.

Exercice 3 - Écriture

Un exercice d'écriture. Le patient doit parvenir à réécrire le mot qui est affiché à l'écran, sans fautes.

Comporte plusieurs niveaux de difficulté.
